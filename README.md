The following is part 1 of a 3 part project that aims to create a messaging application. The First 
part which is the part that is currently uploaded is the registration and loging feature of the application. 
This allows the user to input a short username and complex password as well as a registred South African phone number.

Since this is part 1 of this project there are no relivent changes so I will just list what was implimented:

- Registration feature - this was implimented using the Scanner object to gain user input and boolean expersions contained in methodd to deal with any inncorct or misinputs
- Login feature - This also user the Scanner object to gain user input and more methods account for inncorect login information or misinputs. The system will aslo display what the problem is andd how to fix it accordingly
- Junit test - A JUnit test was also added for automated testing to ensure that the code works properly
